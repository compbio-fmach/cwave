cwave - Continous Wavelet Transform (CWT) library
=================================================

cwave is a Python2/3 library implementing the Continuous Wavelet Transform (CWT)
using FFT as in `A Practical Guide to Wavelet Analysis` by C. Torrence and G.P.
Compo.
